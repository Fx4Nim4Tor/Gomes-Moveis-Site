# Gomes-Moveis-Site
Desenvolvimento de Site profissinal para loja de moveis para escritorios.
Site tem o foco de nao ser apenas um site bonito mas sim um util para coleta de informações com objetivo de melhor conversão de venda.
tipo de infors que deve dar para o ADM:
- Quais as categorias mais procuradas;
- Mapa de calor do site(onde o usuario mais clica)
- Quantos usuarios voltam para a pagina;
- Qual o publico mais entra (F/M)
- (...)
